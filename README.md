# Retail-Customer-Analytics
End-to-end Retail Customer Analytics project using Python, PostgreSQL, and Power BI. Analyzed 3,900 customer transactions to uncover purchasing patterns, customer segments, subscription trends, and revenue insights through SQL analysis and interactive dashboards.

## Project Overview

This project analyzes customer shopping behavior using Python, PostgreSQL, and Power BI to uncover purchasing patterns, customer segments, subscription trends, and revenue drivers. The goal is to transform raw retail transaction data into actionable business insights that support data-driven decision-making.

## Business Problem

Retail businesses need to understand customer purchasing behavior across demographics, product categories, discounts, subscriptions, and shopping channels. This project helps identify customer trends, improve engagement strategies, optimize product offerings, and increase customer loyalty.

## Dataset Information

* Records: 3,900
* Features: 18
* Domain: Retail / E-commerce
